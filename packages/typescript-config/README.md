# `@workspace/typescript-config`

Shared typescript configuration for the workspace.

## Available Configs

- `base.json` - Base configuration with strict TypeScript settings
- `react-library.json` - Extends base.json, adds React JSX support
- `vite-react.json` - Extends react-library.json, optimized for Vite + React applications
- `nextjs.json` - Extends base.json, optimized for Next.js applications
