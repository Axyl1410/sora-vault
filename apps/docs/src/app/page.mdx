# Get started

Welcome to the documentation for your **e-commerce** monorepo.

This site is powered by the **Axyl Docs** experience (built on Nextra) and uses the shared layout (navbar, sidebar, footer) defined in `layout.js`.

## What’s inside this monorepo?

- **`apps/dapp`** – dapp sora vault
- **`apps/docs`** – This documentation site
- **`packages/ui`** – Shared UI components
