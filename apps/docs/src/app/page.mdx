# Get started

Welcome to the documentation for **Sora Vault** monorepo.

This site is powered by the **Axyl Docs** experience (built on Nextra) and uses the shared layout (navbar, sidebar, footer) defined in `layout.js`.

## What's inside this monorepo?

- **`apps/web`** – Main web application (Next.js 15)
- **`apps/docs`** – This documentation site
- **`packages/ui`** – Shared UI components library
- **`packages/sui-contract`** – Sui Move smart contracts
- **`packages/typescript-config`** – Shared TypeScript configurations
